## üöÄ **Getting Started**

### **Prerequisites**
- Node.js 18+ 
- npm or yarn
- Git

### **Quick Start**
```bash
# Clone the repository
git clone https://github.com/your-username/isolation-diaries.git
cd isolation-diaries

# Install dependencies for both frontend and backend
npm run setup

# Start development servers
npm run dev
```

The application will be available at:
- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:3001

### **Demo Mode**
Visit http://localhost:3000 and explore all features instantly with:
- **Anonymous session** - no registration required
- **Sample community data** - real-world examples
- **Full feature access** - try diary mode, emotion analysis, and more
